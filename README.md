## ✨ Overview

**Diamond Freecam** is a highly optimized cinematic camera system built specifically for **Roblox Content Creators**, **YouTubers**, **TikTok Creators**, and **In-Game Photographers** who require complete camera freedom without distracting UI elements.

> 🎬 Create professional cinematic footage directly inside Roblox with complete control over movement, zoom, and camera behavior.

## 🚀 Features

### 🎮 Advanced Camera Control

* 🏃 Adjustable movement speed

* ⚡ Lightweight and optimized performance

### 🔍 Cinematic Zoom System

* 🖱️ Mouse Wheel zoom support

## 🛠️ Installation

### 📥 Quick Setup

1. Copy the contents of `Freecam.lua`
2. Open your preferred Roblox executor
3. Paste the script
4. Execute in-game
5. Wait for the **Diamond Freecam Control Panel** to appear
6. Click **ENABLE FREECAM**
7. Start recording cinematic footage 🎬
